# Smart Eyewear Assistant

## Description
An AI-powered system for visually impaired users that provides:
- Text recognition
- Object detection
- Face recognition
- Medication reminders

## Technologies Used
- Python
- OpenCV
- TensorFlow
- Raspberry Pi

## Features
- Real-time object detection
- OCR for reading text
- Voice alerts

## Future Enhancements
- Mobile app integration
